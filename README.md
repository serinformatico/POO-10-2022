**Requerimientos:**
- IDE - Intellij
- Java 8
- JDK - Amazon Corretto 18
- GIT

**Instrucciones:**
1° Debe situarse dentro de la raíz de tu partición y ejecutar la siguiente instrucción

3° Para mantener tu repositorio local actualizado, ejecuta la siguiente instrucción
    git pull
